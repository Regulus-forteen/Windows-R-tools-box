一款为 Windows 用户打造的现代化、开源、可扩展的工具箱。它集成了从日常维护到专业增强的各类实用工具，旨在让新手用户开箱即用，也让高级用户和开发者能够轻松定制与贡献。

> **🌟 项目愿景**：构建一个纯净、强大、由社区驱动的 Windows 工具生态，**让开源的力量惠及每一位普通用户，同时确保贡献者的智慧结晶不被商业实体无偿独占**。

## ✨ 特性亮点

*   **🎯 用户友好**：简洁直观的图形界面，工具分类清晰，操作简单明了，小白也能轻松上手。
*   **🔌 高度模块化**：采用插件化架构，每个工具都是独立模块，支持热插拔与动态加载。
*   **⚙️ 开源与透明**：核心框架及绝大多数工具完全开源（AGPL v3 协议），代码可审、行为可查，拒绝任何形式的“流氓行为”。
*   **🚀 兼顾新手与高手**：
    *   对**新手**：提供“一键式”解决方案和图形化向导。
    *   对**高手**：开放完整源代码、插件开发接口和命令行调用支持，允许深度自定义。
*   **🌐 可持续生态**：采用“开源核心 + 可选高级服务”模式，确保项目有长期维护和发展的动力。

## 📜 许可证与我们的承诺

本项目采用 **GNU Affero 通用公共许可证 v3.0 (AGPL-3.0)**。我们坚信，开源的力量在于 **共享与回馈**。

### 这对您意味着：

*   **✅ 您可以自由地**：使用、研究、修改本软件，并将其用于个人、商业或公共目的。
*   **✅ 我们鼓励**：fork 本项目、提交问题、发起拉取请求，并与社区分享您创造或改进的工具。
*   **⚠️ 核心要求（来自 AGPL v3）**：
    **如果您修改了本项目，并通过网络（例如 Web 服务、SaaS）向用户提供修改后的服务，那么您有法律义务向所有用户公开您服务端相应部分的完整源代码。**

### 我们为什么选择 AGPL v3？

> 为了保护每一位贡献者的权益。我们欢迎所有人使用我们的劳动成果，但坚决反对任何实体将社区的集体智慧**私有化并以此牟利而不作回馈**。
> AGPL v3 确保：如果一个商业公司使用并改进了我们的工具来构建其服务，那么这些改进必须开源，从而让整个社区受益。**这不是限制，而是为了让开源变得可持续。**

## ❓ 常见问题

**Q: 这和“图吧工具箱”、“360”有什么区别？**
A: 我们汲取了前者的强大和后者（宣称）的易用，但坚持 **开源、透明、无广告、非商业化** 的核心原则。我们的目标是成为社区驱动、真正以用户需求为导向的工具集合。

**Q: 软件是否免费？会有收费功能吗？**
A: **核心工具箱永远免费且开源。** 未来，我们可能会为需要消耗大量服务器资源、提供专业技术支持或高度定制化的 **高级联网服务** 设立可选的付费订阅。但这绝不会影响现有开源工具的功能。

**Q: 我的公司想内部使用/集成，需要付费吗？**
A: **不需要。** AGPL v3 允许内部商业使用。只有当贵公司 **将修改后的本软件作为服务（SaaS）向公司外部用户提供时**，才需要遵守开源义务。详情请参阅许可证。
